## 1 系统概述

### 1.1 编写目的

本文档同时用于指导"学院学生综合服务与党团管理平台"（以下简称"学生综合服务平台"）的使用者，包括管理员（管理老师）与学生两类角色。文档结合本系统的实际功能（智能咨询、党团培养、通知推送、审批证明、成绩分析、模板下载等）展开说明，帮助用户理解系统能力范围、熟悉登录与公共操作，并为后续管理员端、学生端的具体操作章节提供基础。

### 1.2 系统简介

学生综合服务平台是一套面向学校教学管理的信息化平台，实现学生信息管理、教师管理、课程管理、成绩管理以及权限管理等功能。系统采用Web方式运行，用户可通过浏览器访问系统。

### 1.3 系统功能概览

本系统面向"学生综合服务平台"提供以下七大功能模块：
 
**1. 用户与认证模块**
* 账号注册与登录：支持学生端和管理员端身份选择，使用账号密码登录。
* 个人档案查询与更新：学生可修改姓名、学号、班级、专业、年级、电话号码、邮箱等信息；管理员可修改姓名、电话、邮箱。
* 修改密码：所有用户可在个人信息页修改登录密码。
* 基于JWT的身份凭证：登录后自动获取Token，在浏览器会话中保持登录状态。

**2. 知识查询模块：智能咨询（学生端）/知识库管理（管理员端）**
* 关键词检索政策条目：学生端首页及智能咨询页支持输入关键词查询知识库。
* 分类筛选：支持按"全部、证明类、党团类、奖助类、生活类"分类浏览。
* 附件下载：匹配到的政策条目如有附件可直接下载。
* 后台维护：管理员可在知识库管理页新增/编辑政策条目。

**3. 党团流程模块：党团流程（学生端）/流程配置（管理员端）**
* 入党/入团流程切换：学生可在"党团流程"页切换入党流程与入团流程。
* 节点状态可视化：线性展示各节点状态（已完成/进行中/待激活），点击节点可查看详情。
* 材料提交与草稿：支持材料类型选择（思想汇报/谈话记录等）、附件上传、保存草稿和提交审核。
* 草稿与退回管理：草稿可继续编辑或删除；被退回的材料可修改后重新提交。
* 历史记录：展示审批与操作日志。
* 管理员流程配置：管理员可在后台配置流程节点（节点名称、顺序、开始/截止时间、提醒规则等）。
* 管理员审核：在"流程配置"或"审批管理"页审核学生提交的材料（通过或退回补充）。

**4. 通知管理模块：通知公告（学生端）/通知管理（管理员端）**
* 学生端通知列表：按标签（全部、就业、党团、后勤、毕业生、奖助、安全、活动、其它）筛选通知。
* 通知详情查看：点击通知卡片可查看完整正文内容，通知附件可下载。
* 已读标记：点击查看详情后自动标记为已读，支持"全部已读"功能。
* 管理员通知管理：支持新建通知、选择标签与推送年级、上传附件、保存草稿和发布。
* 已发布通知管理：支持查看、批量删除通知。

**5. 证明申请模块：证明申请、模板下载（学生端）/审批管理、模板管理（管理员端）**
* 学生申请提交：选择DOCX模板、填写用途和说明、自动填充模板生成PDF预览后提交。
* 申请记录查看：展示近一年审批留痕，支持查看详情、下载证明文件。
* 草稿箱：未提交的申请可保存为草稿，后续继续编辑提交。
* 管理员审批：在"审批管理"页对待审核申请进行审批（通过或退回补充）。
* 电子证明下载：审批通过后，学生可在申请详情页下载生成的PDF证明文件。

**6. 成绩分析模块：成绩分析（学生端）/培养方案（管理员端）**
* 成绩单上传：支持Excel/CSV/PDF格式，单次不超过30MB。
* 选课建议：根据分析结果生成缺失模块和选课建议。
* 培养方案查看：学生可下载已发布的培养方案文件。

**7. 用户管理模块**
* 管理员端支持查询用户、新建用户和编辑学生信息。

### 1.4 用户角色说明

| 角色 | 功能权限 |
| :--- | :--- |
| 管理员 | 后台管理、知识库维护、通知发布与管理、流程配置、审批管理、模板管理、用户管理、培养方案管理、数据导入导出、个人信息维护 |
| 学生 | 智能咨询、党团流程查看与材料提交、通知查看、证明申请提交与查看、成绩分析、模板下载、个人信息维护 |

## 3 系统登录与退出

### 3.1 用户登录

**操作步骤**：

1. 打开系统后，首先进入登录页面（如未登录，页面仅显示登录界面，顶部导航栏自动隐藏）。
2. 选择登录身份：在下拉框中选择"学生端"或"管理员端"。
3. 输入账号：学生端测试账号为 `stu001`，管理员端测试账号为 `admin001`。
4. 输入密码：测试账号密码为 `123456`。
5. 点击【登录】按钮提交。
6. 登录成功后，页面自动跳转至对应的系统首页（学生端显示学生首页，管理员端显示后台首页）。

**登录界面**
![登陆界面](\CustomImg\login.png "登陆界面")

**注意事项**：
* 登录身份选择错误会导致登录失败，提示"身份不匹配"。
* 密码错误时系统会提示"账号或密码错误"。
* 登录状态保存在浏览器的SessionStorage中，关闭浏览器标签页后需重新登录。

### 3.2 用户注册

**操作步骤**：

1. 在登录页面点击【没有账号？点击注册】链接。
2. 在注册表单中选择注册身份（学生端/管理员端）。
3. 填写学号/工号、真实姓名、密码及确认密码。
4. 点击【注册】按钮提交。
5. 注册成功后系统提示"注册成功！请登录"，自动返回登录页面。

**注册界面**
![注册界面](/CustomImg/createAccount.png)

**注意事项**：
注册后需使用注册时填写的账号和密码进行登录。如学号/工号已被注册，系统会提示"该学号/工号已被注册"。

### 3.3 用户退出

**操作步骤**：

1. 在系统界面左侧导航栏底部，点击【退出登录】按钮。
2. 系统自动清除登录状态，返回登录页面。
3. 同时会清除浏览器会话中的所有已保存状态（Token、用户信息、当前浏览视图等）。

## 4 公共功能说明

### 4.1 学生端首页

**功能说明**：
学生登录成功后，默认进入学生端首页。首页主要展示以下内容：

**顶部指标卡片**：
* **待办提醒**：显示党团流程待办事项数量，包括当前节点的材料提交提醒、被退回补充的证明申请等。
* **未读通知**：显示当前未读的通知公告数量。
* **流程进度**：显示入党/入团流程的节点完成进度（已完成数/总节点数）。
* **证明记录**：显示近一年提交的证明申请总数。

**快捷入口卡片**：
* **智能咨询**：进入政策知识库检索页面。
* **党团流程**：进入入党/入团流程查看与材料提交页面。
* **通知公告**：进入通知列表页面。
* **证明申请**：进入证明申请提交与记录查看页面。

**主要内容区**：
* **党团培养进度**：以时间线形式展示各节点状态（已完成/进行中/待激活），点击节点可查看详情（开始时间、截止时间、提醒规则等）。
* **待办提醒**：列出当前需要处理的事项（党团材料提交、证明补交等）。
* **智能咨询快速问答**：支持输入关键词快速查询知识库。
* **最新通知**：展示最近的通知公告，点击可查看详情。

**首页界面**
![](/CustomImg/stuHome.png)

管理员登录成功后，默认进入管理员端（后台）首页。首页主要展示以下内容：

**顶部指标卡片**：

* **近期动态**：展示最近提交的申请记录及其状态。

**首页界面**

### 4.3 个人信息管理

**功能说明**：
用户可在系统的个人信息维护页面查看和修改个人基本信息。

**学生端操作步骤**：
1. 左侧导航栏底部点击【修改个人信息】按钮，或在"证明申请"页面点击【个人信息维护】按钮。
2. 进入个人信息维护页面，可修改的内容包括：
   - 姓名、学号、班级、专业、年级（仅限24级/25级/26级/27级）、电话号码、邮箱。
3. 修改完成后点击【保存信息】按钮提交。
4. 保存成功后系统提示"个人信息更新成功！"。

**学生端修改信息管理界面**
![](/CustomImg/stuInfo.png)

**管理员端操作步骤**：
1. 左侧导航栏底部点击【修改个人信息】按钮。
2. 进入个人信息维护页面，可修改姓名、电话号码、邮箱（账号不可修改）。
3. 修改完成后点击【保存信息】按钮。

**管理员端修改信息管理界面**
![](/CustomImg/adminInfo.png)

**校验规则**：
* 电话号码必须为11位数字。
* 邮箱需符合标准邮箱格式。
* 学生年级仅能从预设选项中选择。
* 学号修改后将同步更新登录账号，下次需使用新学号登录。

### 4.4 修改密码

**功能说明**：
所有用户均可自主修改登录密码，提高账户安全性。

**操作步骤**：
1. 在个人信息维护页面下方找到"账户安全"区域的"修改密码"表单。
2. 填写当前密码、新密码（至少6位）、确认新密码。
3. 点击【修改密码】按钮提交。
4. 修改成功后系统强制退出登录，提示"密码修改成功，请使用新密码重新登录"。

**规则说明**：
* 新密码不能与当前密码相同。
* 两次输入的新密码必须一致。
* 新密码长度不能少于6位。
* 修改后需使用新密码重新登录。

**修改密码界面**
![](/CustomImg/changePassword.png)

## 5 管理员功能使用说明

管理员登录成功后，通过左侧导航栏可进入各管理模块。以下按导航顺序逐一说明各功能的使用方法。

### 5.1 用户管理

**功能说明**：
管理员可查询、新建、编辑和删除系统用户账号。用户管理支持对学生角色（普通学生、班团骨干）和管理角色（管理老师、学院领导）的统一管理。

**管理界面**
![](/CustomImg/accountManage.png)

**查询用户**：
1. 在左侧导航栏点击【用户管理】。
2. 页面展示所有用户的表格列表，包含账号、姓名、角色、专业、年级、电话、邮箱、状态等字段。
3. 在搜索框中输入账号、姓名、专业或年级关键词，点击【查询】按钮即可筛选符合条件的用户。

**新建用户**：
1. 若先前正在编辑用户，需要点击取消编辑按钮。
2. 在用户管理页右侧的"账号信息"表单中填写用户信息。
3. 填写内容：账号（必填）、初始密码（必填）、姓名、角色（普通学生/班团骨干/管理老师/学院领导）、状态（启用/停用）、电话、邮箱、专业、年级、部门。
4. 点击【创建用户】按钮提交。
5. 创建成功后列表自动刷新。

**编辑用户**：
1. 在用户列表中点击目标用户行的【编辑】按钮。
2. 右侧表单自动载入该用户的信息，姓名、专业、年级、电话、邮箱等字段可修改。
3. 如需重置密码，在"重置密码"字段输入新密码。
4. 点击【保存用户】按钮提交修改。

**删除用户**：
1. 在用户列表中点击目标用户行的【删除】按钮。
2. 系统弹出确认对话框，确认后该用户及其关联数据（申请记录、流程记录、通知投递记录等）将被一并删除。

**注意事项**：
* 新建用户时初始密码必须填写。
* 编辑用户时如不修改密码，可留空"重置密码"字段。
* 学生角色可填写专业和年级，管理角色可填写部门。

### 5.2 知识库管理

**功能说明**：
管理员可新增、编辑政策条目，维护知识库内容。知识库条目支持标题、分类、关键词、正文内容和附件上传，供学生端智能咨询模块检索和展示。

**管理界面**
![](/CustomImg/policyManage.png)

**新增条目**：
1. 在左侧导航栏点击【知识库管理】。
2. 点击页面右上角的【新增条目】按钮。
3. 在右侧"政策内容"表单中填写：
   - 标题：政策条目标题（如"在读证明开具说明"）。
   - 分类：选择"证明类、党团类、奖助类、生活类、学业类"中的一类。
   - 关键词：输入多个关键词用逗号分隔（如"在读证明, 下载, 模板"）。
   - 内容：填写标准答复、办理说明、注意事项等正文。
   - 附件上传：可选上传PDF/Word/Excel等附件，单个文件不超过30MB。
   - 状态：选择"启用"或"草稿"。
4. 点击【保存条目】按钮提交。

**编辑条目**：
1. 在知识库列表中点击目标条目的【编辑】按钮。
2. 右侧表单自动载入该条目的标题、分类、关键词、内容等信息。

## 7 常见问题与解决方法

**问题1：无法登录系统**

**可能原因**：
* 用户名或密码错误
* 选择的登录身份与实际账号身份不匹配

**解决方法**：
* 检查账号和密码是否正确，测试账号：学生端 `stu001`，管理员端 `admin001`，密码 `123456`
* 确认选择的登录身份与账号身份一致

**问题2：党团流程中无法提交材料**

**可能原因**：
* 当前流程节点尚未到开始时间
* 已超过节点截止时间

**解决方法**：
* 在节点详情中查看开始与截止时间，在开放时间内提交
* 超过截止时间时仍可保存草稿，但无法正式提交


**问题3：证明申请中无法正确生成pdf预览**

**可能原因**：
* 选择的模板缺少必要的占位符（如{姓名}、{学号}等关键字段）
* 模板中使用的是普通文本而不是系统可识别的占位符

**解决方法**：
* 联系管理员检查模板是否包含标准占位符
